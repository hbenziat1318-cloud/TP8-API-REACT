# TP React - Consommation d'API avec Fetch et Axios

##  Description

Ce projet React démontre la consommation d'API REST en utilisant deux méthodes différentes : **Fetch API** (native au navigateur) et **Axios** (librairie externe). L'application récupère et affiche des données depuis JSONPlaceholder, une API REST gratuite pour les tests.

##  Fonctionnalités

-  **Récupération de données** avec Fetch API
-  **Récupération de données** avec Axios  
-  **Interface moderne** avec design Glassmorphisme
-  **Navigation par onglets** entre Fetch et Axios
-  **Gestion des états** (loading, error, data)
-  **Design responsive** et animations fluides
-  **Boutons de rechargement** des données
-  **Cartes interactives** avec effets hover
-  **Indicateurs visuels** élégants


##  Installation et Démarrage

bash
### Cloner le dépôt
git clone https://github.com/hbenziat1318-cloud/TP8-API-REACT.git

### Se déplacer dans le dossier
cd TP8-API-REACT

### Installer les dépendances
npm install

### Démarrer l'application
npm start


L'application sera accessible sur : `http://localhost:3000`
